# 回溯算法
回溯算法是纯暴力的搜索

## 分析三部曲
- 递归的参数和返回值
- 确定终止条件
- 单层搜索的逻辑（一般为for循环， 遍历的是集合）


## 递归和回溯相辅相成
- 递归函数的下面就是回溯的过程


## 回溯算法解决的问题
- 组合问题（不强调组合的顺序）
- 切割问题
    - 通常给定一个字符串问有几种切割方式
    - 或者加一下特定的切割条件
- 子集问题
- 排列问题（强调组合顺序）
- 棋盘问题


## 回溯模板（解题步骤）
- backtracking(参数)
- if (终止条件) {收集结果； return}
- for (集合元素) {处理节点；递归函数；回溯操作；} return
